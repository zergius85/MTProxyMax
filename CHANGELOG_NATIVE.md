# 📝 Changelog MTProxyMax Native

История изменений версии Native (без Docker).

---

## v1.0.4-native (2026-03-28)

### ✨ Новые возможности

#### 🔹 Полная работа без Docker
- Нативный запуск telemt напрямую в системе
- systemd сервисы для всех компонентов
- Потребление RAM снижено с ~50 MB до ~30 MB

#### 🔹 Русский язык
- Полная локализация интерфейса
- Файлы переводов `locales/ru.json` и `locales/en.json`
- Модуль `locale.sh` для загрузки переводов

#### 🔹 Веб-интерфейс (FastAPI)
- **Дашборд** — статус прокси, статистика, управление
- **Страница секретов** — добавление, редактирование, удаление
- **Страница настроек** — порт, домен, Ad-Tag, маскировка
- **Страница логов** — просмотр в реальном времени, фильтр, скачивание
- Адаптивный дизайн в тёмных тонах
- Автообновление логов каждые 5 секунд

#### 🔹 Telegram-бот с inline-меню
- **Главное меню** с кнопками
- **Мастер добавления секрета** — пошаговый ввод с валидацией
- **Управление секретами** — включение, отключение, вращение, удаление
- **Просмотр ссылок** — копирование в буфер
- **Управление прокси** — старт, стоп, рестарт, логи
- **Настройки** — изменение параметров

### 🔧 Улучшения

#### CLI
- Упрощённая установка через `install_native.sh`
- Улучшенные сообщения об ошибках
- Валидация входных данных

#### Веб-интерфейс
- Безопасный парсинг конфигов
- Автоопределение публичного IP
- Генерация ссылок в формате FakeTLS
- Улучшенная обработка ошибок API

#### Telegram-бот
- Пошаговый мастер создания секретов
- Валидация имени, лимитов, дат
- Развёрнутые сообщения об ошибках
- Inline-кнопки для быстрого доступа

### 📁 Структура проекта

```
MTProxyMax/
├── engine/
│   ├── install_engine.sh     # Установка telemt
│   └── enginectl.sh          # Управление движком
├── locales/
│   ├── ru.json               # Русский перевод
│   └── en.json               # English
├── web/
│   ├── app.py                # FastAPI сервер
│   ├── requirements.txt      # Python зависимости
│   ├── templates/
│   │   ├── index.html        # Дашборд
│   │   ├── secrets.html      # Управление секретами
│   │   ├── settings.html     # Настройки
│   │   └── logs.html         # Логи
│   ├── static/               # CSS, JS
│   └── mtproxymax-web.service
├── bot/
│   ├── bot.py                # Telegram бот
│   ├── requirements.txt      # Зависимости
│   ├── .env.example          # Шаблон конфига
│   └── mtproxymax-bot.service
├── docs/
│   └── README.md             # Полная документация
├── locale.sh                 # Система локализации
├── mtproxymax-native.sh      # Главный скрипт
├── install_native.sh         # Полный установщик
├── README_NATIVE.md          # Быстрый старт
└── CHANGELOG_NATIVE.md       # Этот файл
```

### 🐛 Исправления

- Исправлен парсинг числовых значений в secrets.conf
- Улучшена обработка ошибок API
- Исправлено отображение ссылок в Telegram
- Устранены проблемы с кодировкой в логах

### 📚 Документация

- `docs/README.md` — полная документация (300+ строк)
- `README_NATIVE.md` — быстрый старт
- `CHANGELOG_NATIVE.md` — история изменений
- `INSTALL.md` — подробная инструкция по установке

### 🔒 Безопасность

- Валидация имени секрета (только латиница)
- Проверка дубликатов
- Ограничение длины имени (32 символа)
- Безопасный парсинг конфигов (без eval/source)
